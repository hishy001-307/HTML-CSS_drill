# Gym-03：Flex でレイアウトを組んでみよう（2 カラム + カード一覧）

## このジムの目的

このジムでは、Web 制作で最頻出のレイアウトを作ります。

- **2 カラム**（サイドバー + メイン）
- **カードを複数並べる**（折り返しあり）

<!-- ここまでできると、次のボス（YouTube 風 UI）にそのままつながります。 -->

---

## やること

`index.html` と `style.css` を編集して、  
**管理画面っぽいレイアウト**を完成させてください。

---

## 変更する場所（重要）

CSS 内にある

- `↓↓↓ ここを編集してください ↓↓↓`

と書いてある場所を編集してください。

HTML 側では

- ページ見出し（Dashboard）
- カードを 6 枚以上に増やす（複製する）

を実施してください。

---

## 達成条件（ここまでできれば OK）

- [ ] サイドバーが左、メインが右の 2 カラムになっている（Flex）
- [ ] メインは残り幅を使って広がる（`flex: 1`）
- [ ] カードが横に並び、狭い画面では折り返す（`flex-wrap`）
- [ ] カードが最低 6 枚ある

---

## ヒント

- 2 カラムは「親を Flex」にすると一気にできる
- `flex: 1` は「残り全部を使う」最重要ワード
- 折り返しは `flex-wrap: wrap`

<aside>の<a>の中身はいじらなくてOKです。
