# Gym-02：カード UI を作ってみよう（ボックスモデル）

## このジムの目的

このジムでは、CSS の超重要概念である **ボックスモデル** を体で覚えます。

- `width / height`（箱の大きさ）
- `padding`（箱の内側の余白）
- `border`（箱の枠線）
- `margin`（箱の外側の余白）
- `box-sizing`（幅の計算を楽にするおまじない）

結論：UI はだいたい **「箱」** です。まず箱を作れれば勝ちです。

---

## やること

`starter/index.html` と `starter/style.css` を編集して、  
**プロフィールカード（1 枚）** を完成させてください。

## 達成条件（ここまでできれば OK）

- [ ] カードが中央に配置されている（`margin: 0 auto`）
- [ ] カードに `padding` があり、文字が端にくっついていない
- [ ] カードに `border` があり、「箱」に見える
- [ ] ボタンがボタンっぽく見える（`padding` と `border` と `display`）

---

## 余力があれば（任意）

- [ ] `border-radius` を調整して、角丸の気持ちよさを探す
- [ ] `hover` を有効にして、マウスを乗せた時に変化させる
- [ ] カードに影を付けてみる（`box-shadow`）
