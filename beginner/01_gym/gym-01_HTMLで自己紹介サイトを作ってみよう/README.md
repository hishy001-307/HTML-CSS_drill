Gym-01：HTML で自己紹介ページを作ってみよう

このジムの目的

このジムでは、HTML / CSS のいちばん基本的な役割を確認します。
• HTML：
👉 文章や情報の構造を作るもの
• CSS：
👉 見た目を整えるもの

まだ「かっこいいデザイン」を作る必要はありません。
まずは 正しいタグを、正しい意味で使えるようになること がゴールです。

⸻

やること

Edit_me!の中にある starter/index.html と starter/style.css を編集して、
シンプルな自己紹介ページを完成させてください。

HTML でやること

以下の要素をすべて使います。
• 見出し（h1, h2）
• 文章（p）
• 箇条書き（ul, li）
• リンク（a）
• 画像（img）

👉 すでに ひな形は用意してあります。
↓↓↓ ここを書き換えてください ↓↓↓ と書いてある場所だけ編集してください。

⸻

CSS でやること

CSS では、次の点を確認します。
• 文字サイズが変えられる
• 行間（line-height）を調整できる
• 余白（margin, padding）で読みやすさが変わる
• 画像が画面からはみ出ないようにできる

👉 すでに書いてある CSS の 数字をいじるだけで OK です。
「何が変わるか」を目で見て確認してください。

チェックリスト（できたか確認）
• HTML のコンテンツを変えて自分
• CSS の数字を変えると見た目が変わることを確認した

⸻

ヒント
• HTML は「文章の構造」を決めるだけでいい
• CSS は「読みやすくする装飾」くらいの気持ちで OK
• 分からなくなったら、テキストを読み返そう

⸻

このジムが終わったら、次は 「箱（ボックス）」 を扱います。
• div が登場します
• 余白・枠線・カード UI を作ります
